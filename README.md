# Andrew Craeton — Portfolio (React + TypeScript)

A single-page application (SPA) personal portfolio built with **React + TypeScript (Vite)** to showcase my skills, projects, experience, education, and certifications.

This site is designed to be fast, responsive, and easy to update by editing a single data file.

---

## Live Site

- [GitHub Pages](https://craetona.github.io/drew-portfolio/)

---

## What’s Included

- **About / Summary** pulled directly from my resume
- **Skills** (technical + soft)
- **Timeline tree** for all dated items:
  - Experience
  - Education
  - Projects
  - Certifications
- **Contact section** with email/phone and links (LinkedIn/GitHub)

---

## Tech Stack

- React
- TypeScript
- Vite
- Plain CSS (no UI framework)
